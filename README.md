# NorthStar Urban Mobility and Logistics Analytics Project

## Overview
This project analyses operational inefficiencies within NorthStar Urban Mobility and Logistics using SQL in R, R analytics, Python data processing, and MongoDB Atlas.

## Technologies Used
- Python
- Pandas
- NumPy
- R
- SQL
- MongoDB Atlas
- Google Colab

## Project Objectives
- Analyse customer complaints
- Investigate operational inefficiencies
- Explore app-event behaviour
- Design a NoSQL database solution
- Apply query optimisation techniques
